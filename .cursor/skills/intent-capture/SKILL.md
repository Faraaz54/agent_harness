# Intent Capture

Intent contains:

1. Goal — one outcome, implementation-neutral.
2. Constraints — qualities the outcome must carry.
3. Failure Conditions — binary failure checks owned by validation.
4. Explicit Non-Goals.
5. Human Notes.

Do not turn the intent into a spec. If the goal contains “and”, split it.
